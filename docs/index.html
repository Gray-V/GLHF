About

Hey gamers! You ever feel like a bot frag duelist on a 7-day lose streak while programming video games? Worry not keyboard warriors, skip the grind and level up your coding game, gamer! GLHF is an OP legendary drop that will boost your coding stats!

GLHF is a coding language made by gamers, for gamers. It streamlines and simplifies all the tools a game developer would commonly use! It includes built in gameloops, player controlers, and window instantiators! Now, making games is as simple as playing them!

Math


Function add(x,y) --untyped function
    return(x+y) --return statement
end add

Function multiply(x,y)
    return(x*y)
end multiply

add(1,2) --function call
multiply(3,5)

print(5/2) --division
print(2-2) --subtraction

print(2**2) --exponential evaulates to 4
print(-2**2) --exponential evaulates to 4

print(3%2) --modulos evaulates to 1
